
**EcoShare** — лёгкая и безопасная веб‑платформа для бесплатного обмена вещами. Приложение работает полностью в браузере и не требует регистрации: пользователи быстро публикуют объявления и находят нужные предметы в своём сообществе.

**Цель**
- Снизить количество отходов за счёт повторного использования вещей.
- Упростить обмен предметами между соседями и знакомыми.
- Сделать процесс отдачи и получения вещей быстрым и прозрачным.

**Ключевые особенности**
- Простая форма добавления объявления: заголовок, описание, контакты, изображение (опционально).  
- Моментальное отображение объявлений в виде карточек.  
- Локальное хранение данных в `LocalStorage` — данные остаются на устройстве пользователя.  
- Адаптивный интерфейс, удобный на мобильных и десктопных устройствах.  
- Минималистичный дизайн, акцент на читаемость и удобство взаимодействия.

**Как это работает (кратко)**
1. Пользователь заполняет форму на странице «Добавить вещь» и при желании загружает фото.  
2. Данные сохраняются в `LocalStorage` в виде массива объектов.  
3. На странице «Объявления» скрипт читает `LocalStorage` и рендерит карточки с информацией и контактами.  
4. При очистке данных браузера объявления удаляются (локальное хранение).

**Кому полезно**
- Людям, которые хотят отдать вещи бесплатно.  
- Тем, кто ищет недорогие или бесплатные предметы поблизости.  
- Сообществам и инициативам по сокращению отходов и повторному использованию.

**Технические детали (коротко)**
- Формат хранения: JSON‑массив объектов под ключом, например, `eco_ads`.  
- Изображения при загрузке кодируются в base64 и сохраняются вместе с объявлением.  
- Для многопользовательской работы и централизованного хранения можно подключить бэкенд и базу данных (опционально).

**Примечание**
Проект рассчитан на локальное использование и быстрый старт; при необходимости его можно расширить серверной частью, авторизацией и поиском по объявлениям.
