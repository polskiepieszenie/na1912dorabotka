
# О проекте — EcoShare

**EcoShare** — лёгкая и безопасная веб‑платформа для бесплатного обмена вещами.  
Проект работает полностью в браузере и не требует регистрации: пользователи быстро публикуют объявления и находят нужные предметы в своём сообществе.

---

## Миссия
- Снизить количество отходов за счёт повторного использования вещей.  
- Упростить обмен предметами между соседями и знакомыми.  
- Сделать процесс отдачи и получения вещей быстрым и прозрачным.

---

## Как это работает
1. Пользователь заполняет форму на странице **Добавить вещь** и при желании загружает фото.  
2. Данные сохраняются в `LocalStorage` в виде массива объектов.  
3. На странице **Объявления** скрипт читает `LocalStorage` и рендерит карточки с информацией и контактами.  
4. При очистке данных браузера объявления удаляются (локальное хранение).

---

## Ключевые особенности
- Простая форма добавления: заголовок, описание, контакты, изображение (опционально).  
- Моментальное отображение объявлений в виде карточек.  
- Локальное хранение данных в `LocalStorage`.  
- Адаптивный интерфейс для мобильных и десктопных устройств.  
- Минималистичный дизайн с акцентом на удобство.

---

## Технические детали
- Данные хранятся под ключом `eco_ads` в формате JSON (массив объектов).  
- Изображения кодируются в base64 при загрузке и сохраняются вместе с объявлением.  
- Для централизованного хранения и многопользовательской работы можно подключить бэкенд и базу данных.

---

## Планы развития
- Поиск и фильтрация объявлений.  
- Экспорт/импорт объявлений (JSON).  
- Авторизация и профиль пользователя.  
- Хранение изображений в облаке (например, Firebase Storage).
